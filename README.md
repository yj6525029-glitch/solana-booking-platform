# Solana AI Booking Agent

AI-powered hotel booking platform using Solana blockchain to eliminate overbooking and payment errors.

## Problem
- Hotel overbooking due to sync failures
- Payment errors with manual processing
- Poor customer support availability
- High commissions (15-20%)

## Solution

| Feature | Technology |
|---------|------------|
| AI Natural Language Search | LLM-powered booking queries |
| NFT Room Inventory | Metaplex cNFTs prevent double-booking |
| Smart Contract Escrow | Auto-refunds, instant payouts |
| Solana Pay | Fast, low-cost transactions |

## Quick Start

```bash
npm install
npm run dev
```
